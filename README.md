# Coding style rules

## 命名
* 變數: 駝峰 ex.$fontSmall
* 屬性(選擇器): 底線 ex.#form_field

## 縮排
* Spaces: 4

## 換行
* 遇{}請換行 ex.
    function(){
        return;
    }

## Font-awesome 的格式
*如果是在 main.js 要使用 icon，就要使用 camelCase ，但在 App.vue 則要使用 kebab-case。
*Note that you add the icons in camelCase in main.js but in kebab-case in App.vue.
